# Zaira's World - Shoe Inventory Management System

A multi-tenant, mobile-responsive shoe inventory management system built with Angular and .NET 8 Web API.

## Features
- **Supplier Directory**: Manage vendor details.
- **Shoe size configurations**: Configure and check stock levels by sizes.
- **Product Entry Form**: Add items with size-quantity matrix mapping.
- **Automated Barcodes**: Visual Code 128 barcode generator.
- **CI/CD Pipeline**: GitHub Actions auto-deployment to Linux servers.
